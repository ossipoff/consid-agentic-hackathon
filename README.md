# Consid Agentic Hackathon

AI agent projects built during the Consid Agentic Hackathon.

## Projects

### [SogneAgent](./SogneAgent/)

A Danish Parish Information Agent built with .NET 8.0 and Microsoft Semantic Kernel. Interactive chatbot that queries the Dataforsyningen API for Danish parish data using automatic LLM function calling.
